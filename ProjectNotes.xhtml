You might need to do this when setting up an IDEA project imported from eclipse:
    figure out where the run dir is, for example:
      cd ~/Library/Caches/IntelliJIdea10/gwt/anarchia.anarchiaa6c87dc1/Anarchia.b093d17e/run/www
      cp ~/src/anarchia/war/AnarchiaEGWT.html .
    When IDEA sets up the sample app, it copies this over as part of dev mode deployment, but
    on importing projects, seems to skip this step.
